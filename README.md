# File-Explorer 文件资源管理器

一个设计优良部署简单使用PHP编写的文件管理器，所有功能集成于一个文件.

:示例链接(EN): [Live Demo](https://web-file-explorer.herokuapp.com/)



## 安装说明

* 你只需要将 [`explorer.php`](https://github.com/webcdn/File-Explorer/raw/standalone/explorer.php) 复制/上传到你的网络服务器的根目录
* 检查一次所需的php配置 设置新密码进行访问
* 仅此而已

**可选:** 你可以将`explorer.php`重命名为`index.php` 你懂我的意思



## 功能说明

* 单文件系统 不需要其他文件(可以在`localhost`中使用)
* 艺术品般的UI界面
* 完全基于Ajax的快速响应
* 符合我(原作者)的审美观 更像是Google Drive 而不是Windows Explorer.
* 允许拖放或浏览文件上传
* 压缩和解压zip文件
* 简单的文本/代码编辑器
* UI/UX支持
* XSRF保护
